\# WordIndexer



> Empowering authors, researchers, educators, publishers, and technical writers with intelligent, open-source tools for analyzing, indexing, and improving Microsoft Word manuscripts.



\---



\## Mission



WordIndexer is an open-source platform dedicated to helping writers produce better books and documents.



Our mission is to empower authors, researchers, educators, publishers, and organizations with intelligent tools that automate indexing, analyze manuscript quality, generate reports, and simplify the publishing workflow.



Our first goal is automatic Microsoft Word index generation using native XE fields while preserving document formatting.



Over time, WordIndexer will evolve into a complete manuscript analysis platform.
