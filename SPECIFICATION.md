\# WordIndexer Specification



\## Project



WordIndexer is an open-source engine for automatically generating Microsoft Word indexes by inserting XE fields into `.docx` documents.



\---



\## Version 1 Objectives



\- Load a Microsoft Word document

\- Read one or more JSON dictionaries

\- Locate every occurrence of indexed terms

\- Ignore the Table of Contents

\- Resolve overlapping matches

\- Insert XE fields

\- Save an indexed document



\---



\## Supported Input



\- Microsoft Word (.docx)



\---



\## Supported Output



\- Microsoft Word (.docx)



\---



\## Dictionary Format



JSON dictionaries containing:



\- metadata

\- entries

\- aliases

\- categories

\- canonical index terms
